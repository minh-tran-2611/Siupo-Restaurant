# 🎯 Project Overview

## Introduction

**Siupo Restaurant** is a modern, full-featured restaurant web application built with React 19 and TypeScript. The application provides a seamless user experience for browsing menus, placing orders, table reservations, and managing user accounts.

---

## 🌟 Key Features

### Customer Features

- **🍽️ Menu Browsing**
  - Browse restaurant menu with categories
  - Advanced filtering and sorting
  - Detailed product information
  - High-quality product images

- **🛒 Shopping Cart**
  - Add/remove items
  - Real-time quantity updates
  - Price calculations
  - Apply discount coupons

- **📅 Table Reservation**
  - Book tables for specific dates and times
  - Select number of guests
  - Pre-order menu items
  - Booking confirmation

- **💳 Checkout & Payment**
  - Multiple payment methods
  - Address management
  - Order summary
  - Secure payment processing

- **📦 Order Management**
  - View order history
  - Track order status
  - Order details and receipts
  - Reorder functionality

- **❤️ Wishlist**
  - Save favorite items
  - Quick add to cart
  - Manage saved items

### User Account

- **👤 Authentication**
  - Email/password registration
  - OAuth2 (Google) login
  - Password recovery
  - OTP verification

- **⚙️ User Settings**
  - Profile management
  - Address book
  - Order history
  - Account preferences

### Multi-language Support

- **🌐 i18n**
  - English (EN)
  - Vietnamese (VI)
  - Dynamic language switching
  - 12 translation namespaces

### Design & UX

- **📱 Responsive Design**
  - Mobile-first approach
  - Tablet and desktop optimized
  - Touch-friendly interactions

- **🎨 Modern UI**
  - Material-UI components
  - Smooth animations (Framer Motion)
  - Clean and professional design
  - Consistent color palette
